== Costs Manager

Програма контролю доходів та витрат.

Програма наявна у двох варіантах: online проект та desktop аплікація.

Desktop версію програми можна скачати із сайту, що забезпечує даний сервіс -
це може бути готовий інсталятор програми або просто арчів із необхідними файлами проекту.

Як і на сайті, для входу в програму слід буде створити обліковий запис користувача або 
ввести логін та пороль уже зареєстрованого в системі.

Ядром сервісу є база даних, де зберігається вся інформація занесена користувачем протягом 
всього часу користування сервісом/проограмою. Програма надає доступ до даниз, що є в базі,
представляє їх у наглядному та зрозумілому користувачу вигляді: графіки, діаграми, таблиці.
Додатково, на основі головної бази даних,що прив'язана до web аплікації та зберігається на 
сервері, можна генерувати деяку статистику витрат всіх користувачів, що користуютися таким 
сервісом (online проект та desktop аплікація).

Програма містить деякі вбудовані (дефолтні) категорії доходів та витрат (що є бльш-менш у всіх 
людей), наприклад витрати на харчування, одяг/взуття, відпочинок.поїздки у громадському транспорті 
(власний транспорт - пальне для авто, його обслуговування); надходження зар. плати (стипендії 
для студентів).Крім того користувач може додавати нові категорій та змінювати (перейменовувати, 
знищувати) деякі з існуючих. Для того щоб додати інформацію про витрати/доходи можна передбачити 
кілька варіантів: 
*) додавання інформації на форму, що представляє поточний день;
*) додавання інформації для одноразового отримання/трати грошей;
*) або вибір деякої категорії - з'являється табличка по поточнії категорії, куди можна додати новий рядочок.
...

Далі користувач може отримти інфорацію по доходах/витратах за деякий проміжок часу (кілька днів. тижнів, місяці, рік)
+ передбачити два календарики для вибору користувачем дня початку та кінця виписки самостійно. Інформацію можна представляти 
у вигляді таблиць (по замовчуванню), графіків, діаграм.

Є ще опція встановлення деякого планованого ліміту (обмеження) на витрати по деякій категорії.
Так, скажімо, можна розпланувати орієнтовні витрати на тиждень, місяць по конкретним категоріям.
О, ще можна організувати таку опцію, щоб коли витрати по категорії зашкалюють (просто завеликі
чи перевищують встановлений ліміт), то при завантаженні програми з'являлося маленьке повідомлення
з попередженням, клацнувши на яке можна отримати деякі рекомендації (при вході на сайт випливаюче 
вікно з інформацією та рекомендаціями). Та і взагалі при завантаженні програми чи сайту у головному
вікні може з'являтися деяка цікава інформація із загальною статистикою (це для сайту) чи конкретно
для поточного користувача.
